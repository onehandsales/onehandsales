# Front Convention 문서

## 1. 목적

이 폴더는 User Web과 Admin Web의 코드, 주석, 로깅 컨벤션을 관리한다.

## 2. 현재 문서

- `FRONTEND_USER_WEB.md`
- `ADMIN_WEB.md`
- `COMMENT_AND_LOGGING.md`

## 3. 적용 규칙

- User Web 컨벤션은 `FE/user-web`에 적용한다.
- Admin Web 컨벤션은 `FE/admin-web`에 적용한다.
- `FE` 내부 Markdown 문서는 한국어로 작성한다.
- User Web과 Admin Web은 API client와 feature 내부 구현을 공유하지 않는다.
