### 보류 항목 목록

#### 🔵 Core 후보 (출시 전 검토 필요했던 항목)
- 입력 데이터 검증 + 정규화 (전화번호/이메일)
- 모바일 자동 로그인 + 토큰 만료 무중단 갱신
- 에러 메시지 / 토스트 일관성
- 로그 / 모니터링 최소 셋
- 인앱 피드백 채널

#### 🟢 베타 후 확장 후보
- 거래처 그룹 / 태그
- 최근 본 거래처 / 즐겨찾기
- 영업건 단계별 진행 통계 (개인 대시보드)
- 명함 이미지 보관 + 원본 보기
- 엑셀 Import 컬럼 자동 매핑 학습 (D-11 결정 항목과 중복)
- 메모 / 거래처 변경 이력
- PDF 출력 템플릿 (월별 보고용)
- 데이터 자동 백업 다운로드 (월 1회)

#### 🟡 정체성 검토 필요 (거절 가능성 높음)
- 통화 자동 기록 (전화 수신/발신 로그)
- 거래처 위치 지도 표시
- 거래처 생일/기념일 알림
- AI 메모 요약 / 영업 인사이트 추출
- 회사 정보 자동 보강 (사업자번호 → 회사명/주소/업종)
- 다중 사용자 협업 (팀 모드) — B2C 정체성 정면 위배
- 음성 메모 → 텍스트 변환 (OS 받아쓰기로 대체 가능)
- Slack / 메신저 연동
