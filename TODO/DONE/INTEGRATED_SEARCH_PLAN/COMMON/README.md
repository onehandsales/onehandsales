# COMMON

## 1. 목적

이 폴더는 통합검색 작업에서 Frontend와 Backend가 함께 봐야 하는 사용자 흐름, API 계약, goal 순서, 구현 전 검토 결과를 관리한다.

## 2. 문서 구성

- `USER-FLOW.md`: 사용자가 통합검색을 사용하는 흐름
- `GOAL-WORK-ORDER.md`: `/goal` 작업 순서와 완료 기준
- `API-SPEC/SEARCH_API.md`: `GET /api/search` 계약
- `GOAL-SPECS/G01-BE-INTEGRATED-SEARCH.md`: Backend 구현 goal 상세
- `GOAL-SPECS/G02-FE-INTEGRATED-SEARCH.md`: Frontend 연결 goal 상세
- `PLANNING-REVIEW.md`: 구현 전 기획 검토 결과
