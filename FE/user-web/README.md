# user-web

사용자가 직접 쓰는 Web 앱이다.

## 기술 스택

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui

이 앱은 iOS/Android native app보다 먼저 만드는 첫 MVP 클라이언트다.

## 로컬 실행

전제 조건: Node.js 24 LTS가 활성화되어 있어야 한다.

```bash
pnpm install
pnpm run dev
```

로컬 URL: `http://localhost:5173`

Vercel 프로젝트 root: `FE/user-web`
